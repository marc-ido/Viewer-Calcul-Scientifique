📐 Présentation du problème

Domaine de calcul

Soit \(\Omega = {]{-a},{a}[} \times {]{-b},{b}[}\) un rectangle ouvert de \(\mathbb{R}^2\), où \(a > 0\) et \(b > 0\) sont des paramètres donnés par l'utilisateur. Son bord est défini par :

\[ \partial\Omega = \overline{\Omega} \setminus \Omega = \bigl\{(x,y) \in [-a,a]\times[-b,b] \;\big|\; |x|=a \text{ ou } |y|=b\bigr\}. \]

Équation aux dérivées partielles

On cherche \(u : \Omega \to \mathbb{R}\) solution de :

\[ \text{(1)}\quad -\frac{\partial}{\partial x}\!\left(\eta\,\frac{\partial u}{\partial x}\right) -\frac{\partial}{\partial y}\!\left(\eta\,\frac{\partial u}{\partial y}\right) + \lambda\, u(x,y) = f(x,y) \qquad \text{pour } (x,y)\in\Omega, \]

où \(\lambda > 0\) est une constante réelle strictement positive, et \(\eta, f : \Omega \to \mathbb{R}\) sont deux fonctions continues. On suppose de plus que \(\eta\) est de classe \(\mathcal{C}^1\) sur \(\Omega\) et vérifie :

\[ \exists\,\eta_0 > 0 \text{ telle que } \forall (x,y)\in\Omega,\quad \eta(x,y) \geq \eta_0. \]

Conditions aux limites

Neumann homogène (flux nul aux bords) :

\[ \frac{\partial u}{\partial x}(\pm a, y) = 0,\quad {-b \leq y \leq b} \] \[ \frac{\partial u}{\partial y}(x, \pm b) = 0,\quad {-a \leq x \leq a} \]

Dirichlet homogène (valeur imposée) :

\[ u(x,y) = 0 \qquad \text{pour } (x,y)\in\partial\Omega \]

Le choix se fait via le panneau de contrôle.

🔢 Méthode des Éléments Finis (MEF)

La méthode des éléments finis permet d'approcher la solution \(u\) de ce problème continu par une fonction discrète \(u_h\), calculable numériquement. Voici les grandes étapes :

1
Triangulation du domaine. On découpe \(\Omega\) en \(N \times M\) rectangles puis chaque rectangle est subdivisé en deux triangles, formant un maillage de \(2NM\) triangles et \((N+1)(M+1)\) nœuds. Les nœuds sont aux positions \(\bigl(x_i, y_j\bigr) = \bigl(-a + i\,\tfrac{2a}{N},\; -b + j\,\tfrac{2b}{M}\bigr)\).
2
Fonctions de base barycentriques. Sur chaque triangle \(T\) de sommets \(P_0, P_1, P_2\), on définit trois fonctions \(\varphi_0, \varphi_1, \varphi_2\) (fonctions chapeaux) affines par morceaux telles que \(\varphi_k(P_j) = \delta_{kj}\). Elles forment une base de l'espace d'approximation \(V_h \subset H^1(\Omega)\) : \[ u_h(x,y) = \sum_{k=1}^{(N+1)(M+1)} X_k\,\varphi_k(x,y). \] Les pyramides colorées dans la visualisation 3D représentent ces fonctions : chaque couleur correspond à l'un des trois sommets du triangle.
3
Formulation variationnelle. On multiplie l'EDP par une fonction test \(v \in V_h\) et on intègre par parties sur \(\Omega\). On obtient le problème discret : trouver \(u_h \in V_h\) tel que \[ a(u_h, v_h) = \ell(v_h) \quad \forall v_h \in V_h, \] avec la forme bilinéaire \[ a(u,v) = \int_\Omega \eta\,\nabla u \cdot \nabla v\; \mathrm{d}x\,\mathrm{d}y + \lambda \int_\Omega u\,v\;\mathrm{d}x\,\mathrm{d}y. \]
4
Assemblage de la matrice de rigidité. En substituant \(u_h = \sum_k X_k \varphi_k\) et \(v_h = \varphi_i\), on obtient le système linéaire \(\mathbf{A}\,\mathbf{X} = \mathbf{b}\), où \[ A_{ij} = a(\varphi_j, \varphi_i), \qquad b_i = \int_\Omega f\,\varphi_i\;\mathrm{d}x\,\mathrm{d}y. \] La matrice \(\mathbf{A}\) est creuse, symétrique et définie positive. L'intégration sur chaque triangle est réalisée par une quadrature au point milieu.
5
Résolution par BiCGSTAB. Le système \(\mathbf{A}\,\mathbf{X} = \mathbf{b}\) est résolu par la méthode itérative BiCGSTAB (Biconjugate Gradient Stabilized), adaptée aux matrices creuses non symétriques. La solution \(\mathbf{X} \in \mathbb{R}^{(N+1)(M+1)}\) donne les valeurs nodales \(u_h\) aux nœuds du maillage. La surface colorée dans la visualisation représente cette solution avec une carte de couleurs bleu → cyan → vert → jaune → rouge.

// =============================================================================
//  FEM – Data Structures  (original C++ implementation)
//  Compile with any C++14 compiler; no Emscripten / WebGL required.
// =============================================================================
#pragma once
#include <vector>
#include <functional>
#include <cmath>

// ── Noeud: a mesh node with 2D coordinates ───────────────────────────────────
struct Noeud {
    double x, y;
    Noeud(double x = 0.0, double y = 0.0) : x(x), y(y) {}
};

// ── Triangle: one P1 element – stores 3 global node indices ──────────────────
struct Triangle {
    int i, j, k;
    Triangle(int i, int j, int k) : i(i), j(j), k(k) {}
};

// ── ETA: diffusion coefficient η(x,y) wrapped as a std::function ─────────────
struct ETA {
    std::function<double(double, double)> func;
    ETA() : func([](double, double){ return 1.0; }) {}
    explicit ETA(std::function<double(double, double)> f) : func(std::move(f)) {}
    double operator()(double x, double y) const { return func(x, y); }
};

// ── RHSF: right-hand side f(x,y) ─────────────────────────────────────────────
struct RHSF {
    std::function<double(double, double)> func;
    RHSF() : func([](double, double){ return 1.0; }) {}
    explicit RHSF(std::function<double(double, double)> f) : func(std::move(f)) {}
    double operator()(double x, double y) const { return func(x, y); }
};

// ── VhElement: a P1 triangle with cached node coordinates ────────────────────
struct VhElement {
    int    idx[3];      // global node indices
    double x[3], y[3]; // corner coordinates (cached for performance)
};

// ── Helper: packed index into the 6-entry upper-triangle of a 3×3 matrix ─────
inline int symPacked(int i, int j) {
    // i <= j assumed
    if (i==0 && j==0) return 0;  if (i==0 && j==1) return 1;
    if (i==0 && j==2) return 2;  if (i==1 && j==1) return 3;
    if (i==1 && j==2) return 4;  return 5;
}

// =============================================================================
//  maillageTR – uniform triangular mesh on [-a,a]x[-b,b]
//  Produces (N+1)x(M+1) nodes and 2xNxM triangles.
//  Alternating diagonal pattern: (i+j)%2==0 => NE diagonal, else NW.
// =============================================================================
void maillageTR(double a, double b, int N, int M,
                std::vector<Noeud>&    noeuds,
                std::vector<Triangle>& triangles)
{
    double hx = 2.0 * a / N;
    double hy = 2.0 * b / M;

    // ── Nodes: row-major, j=row (y-direction), i=col (x-direction) ──────────
    noeuds.clear();
    noeuds.reserve((N+1)*(M+1));
    for (int j = 0; j <= M; j++)
        for (int i = 0; i <= N; i++)
            noeuds.push_back({ i*hx - a,  j*hy - b });

    // ── Two triangles per rectangular cell ───────────────────────────────────
    triangles.clear();
    triangles.reserve(2 * N * M);
    for (int j = 0; j < M; j++) {
        for (int i = 0; i < N; i++) {
            int A = (N+1)*j + i;              // bottom-left
            int B = (N+1)*j + (i+1);          // bottom-right
            int C = (N+1)*(j+1) + i;          // top-left
            int D = (N+1)*(j+1) + (i+1);      // top-right
            if ((i+j) % 2 == 0) {
                triangles.push_back({ A, B, D });   // triangle /
                triangles.push_back({ A, D, C });
            } else {
                triangles.push_back({ A, B, C });   // triangle backslash
                triangles.push_back({ B, D, C });
            }
        }
    }
}

// =============================================================================
//  CalcMatBT – Jacobian B_T of the affine map  F_T : T_ref -> T_phys
//
//  B_T = [ x1-x0  x2-x0 ]   (columns are edge vectors from vertex 0)
//        [ y1-y0  y2-y0 ]
//
//  |det B_T| = 2*|T|   (twice the physical triangle area)
// =============================================================================
void CalcMatBT(const VhElement& el,
               double& b00, double& b01,
               double& b10, double& b11)
{
    b00 = el.x[1] - el.x[0];   b01 = el.x[2] - el.x[0];
    b10 = el.y[1] - el.y[0];   b11 = el.y[2] - el.y[0];
}

// =============================================================================
//  integ_eta – approximate  integral_T eta(x,y) dA
//  Midpoint rule at the 3 edge midpoints:
//    integral_T eta dA ≈ ( eta(m01) + eta(m12) + eta(m02) ) / 3 * |T|
// =============================================================================
double integ_eta(const VhElement& el, const ETA& eta)
{
    double b00, b01, b10, b11;
    CalcMatBT(el, b00, b01, b10, b11);
    double absdet = std::fabs(b00*b11 - b01*b10);   // = 2|T|

    double m01x = 0.5*(el.x[0]+el.x[1]),  m01y = 0.5*(el.y[0]+el.y[1]);
    double m12x = 0.5*(el.x[1]+el.x[2]),  m12y = 0.5*(el.y[1]+el.y[2]);
    double m02x = 0.5*(el.x[0]+el.x[2]),  m02y = 0.5*(el.y[0]+el.y[2]);

    double avg = (eta(m01x,m01y) + eta(m12x,m12y) + eta(m02x,m02y)) / 3.0;
    return avg * absdet / 2.0;   // avg * |T|
}

// =============================================================================
//  scdmembre – assemble global RHS  B[k] = integral_Omega f * phi_k dA
//
//  For each element, midpoint quadrature at the 3 edge midpoints.
//  phi_i(midpoint of edge i-j) = 1/2,   phi_i(opposite midpoint) = 0.
//  => local contributions: aire * (f_ij + f_ik) / 6  added to node i.
// =============================================================================
void scdmembre(const std::vector<VhElement>& elements,
               const RHSF& f,
               std::vector<double>& B,
               bool dirichlet,
               const std::vector<bool>& onBnd)
{
    std::fill(B.begin(), B.end(), 0.0);

    for (const auto& el : elements) {
        double b00, b01, b10, b11;
        CalcMatBT(el, b00, b01, b10, b11);
        double aire = std::fabs(b00*b11 - b01*b10) / 2.0;  // |T|

        double m01x=(el.x[0]+el.x[1])/2,  m01y=(el.y[0]+el.y[1])/2;
        double m12x=(el.x[1]+el.x[2])/2,  m12y=(el.y[1]+el.y[2])/2;
        double m02x=(el.x[0]+el.x[2])/2,  m02y=(el.y[0]+el.y[2])/2;

        double f01=f(m01x,m01y), f12=f(m12x,m12y), f02=f(m02x,m02y);

        // phi_0 != 0 on edges 0-1 and 0-2 only
        B[el.idx[0]] += aire * (f01 + f02) / 6.0;
        // phi_1 != 0 on edges 0-1 and 1-2 only
        B[el.idx[1]] += aire * (f01 + f12) / 6.0;
        // phi_2 != 0 on edges 1-2 and 0-2 only
        B[el.idx[2]] += aire * (f12 + f02) / 6.0;
    }

    if (dirichlet)
        for (int k = 0; k < (int)B.size(); k++)
            if (onBnd[k]) B[k] = 0.0;
}

// =============================================================================
//  DiffTerm – local diffusion matrix  K_ij = integral_T eta * grad(phi_i) . grad(phi_j) dA
//
//  Physical gradients: grad(phi) = B_T^{-T} * grad_ref(phi)
//  where  grad_ref(phi_0)=(-1,-1),  grad_ref(phi_1)=(1,0),  grad_ref(phi_2)=(0,1)
//
//  Returns upper triangle packed: K[0..5] = K00,K01,K02,K11,K12,K22
// =============================================================================
void DiffTerm(const VhElement& el, const ETA& eta, double K[6])
{
    double b00, b01, b10, b11;
    CalcMatBT(el, b00, b01, b10, b11);
    double det = b00*b11 - b01*b10;
    if (std::fabs(det) < 1e-30) { for(int i=0;i<6;i++) K[i]=0; return; }

    double val = integ_eta(el, eta);    // approx. integral_T eta dA

    // Rows of B_T^{-1}:  inv = (1/det) * [ b11  -b01 ; -b10  b00 ]
    double inv00= b11/det,  inv01=-b10/det;
    double inv10=-b01/det,  inv11= b00/det;

    // Reference gradients (P1 hat functions on reference triangle)
    static const double gl[3][2] = { {-1,-1}, {1,0}, {0,1} };
    double gr[3][2];   // physical gradients
    for (int i = 0; i < 3; i++) {
        gr[i][0] = inv00*gl[i][0] + inv01*gl[i][1];
        gr[i][1] = inv10*gl[i][0] + inv11*gl[i][1];
    }

    int p = 0;
    for (int i = 0; i < 3; i++)
        for (int j = i; j < 3; j++)
            K[p++] = (gr[i][0]*gr[j][0] + gr[i][1]*gr[j][1]) * val;
}

// =============================================================================
//  ReacTerm – local mass (reaction) matrix  M_ij = lambda * integral_T phi_i * phi_j dA
//
//  Exact P1 formula:  integral_T phi_i * phi_j dA = |T| / 12  (i != j)
//                                                  = |T| /  6  (i == j)
// =============================================================================
void ReacTerm(const VhElement& el, double lam, double Ml[6])
{
    double b00, b01, b10, b11;
    CalcMatBT(el, b00, b01, b10, b11);
    double absdet = std::fabs(b00*b11 - b01*b10);   // = 2|T|

    int p = 0;
    for (int i = 0; i < 3; i++)
        for (int j = i; j < 3; j++)
            Ml[p++] = absdet * lam * (i == j ? 1.0/12.0 : 1.0/24.0);
}

// =============================================================================
//  matvec – global matrix-vector product  W = A * V
//  A = K(eta) + M(lambda), assembled element-by-element (no explicit sparse matrix).
//  Dirichlet BCs: boundary rows replaced by identity (W[k] = V[k]).
// =============================================================================
void matvec(const std::vector<VhElement>& elements,
            const ETA& eta, double lam,
            const std::vector<double>& V,
            std::vector<double>& W,
            bool dirichlet,
            const std::vector<bool>& onBnd)
{
    std::fill(W.begin(), W.end(), 0.0);

    for (const auto& el : elements) {
        double KD[6], KR[6];
        DiffTerm(el, eta, KD);
        ReacTerm(el, lam,  KR);

        for (int i = 0; i < 3; i++) {
            double res = 0.0;
            for (int j = 0; j < 3; j++) {
                int p = (i <= j) ? symPacked(i,j) : symPacked(j,i);
                res += V[el.idx[j]] * (KD[p] + KR[p]);
            }
            W[el.idx[i]] += res;
        }
    }

    if (dirichlet)
        for (int k = 0; k < (int)W.size(); k++)
            if (onBnd[k]) W[k] = V[k];
}

// =============================================================================
//  calcAV – mass-matrix-only product  W = M * V  (lambda=1, no diffusion)
//  Used for normL2 computation (W = M*X, then norm^2 = X^T * W).
// =============================================================================
void calcAV(const std::vector<VhElement>& elements,
            const std::vector<double>& V,
            std::vector<double>& W,
            bool dirichlet,
            const std::vector<bool>& onBnd)
{
    std::fill(W.begin(), W.end(), 0.0);

    for (const auto& el : elements) {
        double Ml[6];
        ReacTerm(el, 1.0, Ml);   // lambda=1, diffusion term omitted

        for (int i = 0; i < 3; i++) {
            double res = 0.0;
            for (int j = 0; j < 3; j++) {
                int p = (i <= j) ? symPacked(i,j) : symPacked(j,i);
                res += V[el.idx[j]] * Ml[p];
            }
            W[el.idx[i]] += res;
        }
    }

    if (dirichlet)
        for (int k = 0; k < (int)W.size(); k++)
            if (onBnd[k]) W[k] = V[k];
}

// =============================================================================
//  bicgstab – solves  A x = b  (BiConjugate Gradient Stabilized)
//
//  Template parameter MatVec: callable  void(const vector<double>& v, vector<double>& w)
//  that computes w = A*v.
//
//  Returns number of iterations taken, or -1 if max iterations reached.
//
//  *** See the "Bridge" tab for the subtle bug that existed in the WASM v12
//  port (but NOT in this original code), and how v13 fixed it. ***
// =============================================================================
template<class MatVec>
int bicgstab(const MatVec& Av,
             const std::vector<double>& b,
             std::vector<double>& x,
             double tol = 1e-8, int maxIter = 2000)
{
    int n = (int)b.size();
    std::vector<double> r(n), r_hat(n), p(n,0), v(n,0), s(n), t(n);

    // r0 = b - A*x0
    Av(x, v);
    for (int i=0;i<n;i++) r[i] = b[i] - v[i];
    r_hat = r;   // arbitrary shadow residual: standard choice r_hat = r0

    double normb = 0;
    for (auto bi : b) normb += bi*bi;
    normb = std::sqrt(normb);
    if (normb < 1e-30) { std::fill(x.begin(),x.end(),0.0); return 0; }

    double rho_old = 1.0, alpha = 1.0, omega = 1.0;

    for (int iter = 0; iter < maxIter; iter++) {

        double rho = 0;
        for (int i=0;i<n;i++) rho += r_hat[i]*r[i];
        if (std::fabs(rho) < 1e-300) break;   // numerical breakdown

        double beta = (iter==0) ? 0.0 : (rho/rho_old) * (alpha/omega);

        // KEY: at this point 'v' holds the OLD A*p from the previous iteration.
        // This is correct -- it will be overwritten by the NEW A*p just below.
        for (int i=0;i<n;i++) p[i] = r[i] + beta*(p[i] - omega*v[i]);

        Av(p, v);   // v = A*p  (overwrites v; next iteration will read it as OLD A*p)

        double rv = 0;
        for (int i=0;i<n;i++) rv += r_hat[i]*v[i];
        if (std::fabs(rv) < 1e-300) break;
        alpha = rho / rv;

        for (int i=0;i<n;i++) s[i] = r[i] - alpha*v[i];

        double norms = 0;
        for (auto si : s) norms += si*si;
        if (std::sqrt(norms) < tol*normb) {
            for (int i=0;i<n;i++) x[i] += alpha*p[i];
            return iter+1;   // early exit
        }

        Av(s, t);   // t = A*s

        double ts=0, tt=0;
        for (int i=0;i<n;i++) { ts += t[i]*s[i]; tt += t[i]*t[i]; }
        if (std::fabs(tt) < 1e-300) break;
        omega = ts / tt;

        for (int i=0;i<n;i++) x[i] += alpha*p[i] + omega*s[i];
        for (int i=0;i<n;i++) r[i]  = s[i] - omega*t[i];

        double normr = 0;
        for (auto ri : r) normr += ri*ri;
        if (std::sqrt(normr) < tol*normb) return iter+1;

        rho_old = rho;
    }
    return -1;   // did not converge
}

// =============================================================================
//  normL2 – discrete L2 norm  ||u_h||^2_{L2} = X^T * M * X
//  Uses calcAV (mass matrix with lambda=1, no BCs).
// =============================================================================
double normL2(const std::vector<VhElement>& elements,
              const std::vector<double>& X)
{
    int n = (int)X.size();
    std::vector<double> MX(n, 0.0);
    std::vector<bool> noBnd(n, false);
    calcAV(elements, X, MX, false, noBnd);   // MX = M*X
    double val = 0;
    for (int i=0;i<n;i++) val += X[i]*MX[i];
    return std::sqrt(val);
}

// =============================================================================
//  normL2Grad – H1 semi-norm  |u_h|^2_{H1} = X^T * K * X  (eta = 1)
//             = integral_Omega |grad(u_h)|^2 dA
// =============================================================================
double normL2Grad(const std::vector<VhElement>& elements,
                  const std::vector<double>& X)
{
    ETA eta_one([](double, double){ return 1.0; });
    double val = 0;

    for (const auto& el : elements) {
        double K[6];
        DiffTerm(el, eta_one, K);
        for (int i = 0; i < 3; i++)
            for (int j = 0; j < 3; j++) {
                int p = (i<=j) ? symPacked(i,j) : symPacked(j,i);
                val += X[el.idx[i]] * K[p] * X[el.idx[j]];
            }
    }
    return std::sqrt(std::fabs(val));
}

// =============================================================================
//  Bridge: Original C++  --->  WASM Adaptation   (what changed and why)
// =============================================================================
//
//  The original code is a standalone C++ FEM library that compiles with any
//  C++14 compiler.  The WASM version runs in the browser via Emscripten /
//  WebGL and needed several practical adaptations.  The mathematics is
//  identical in both versions.
//
// ─────────────────────────────────────────────────────────────────────────────
//  DATA STRUCTURES
// ─────────────────────────────────────────────────────────────────────────────
//
//  Original                          WASM equivalent
//  ──────────────────────────────    ─────────────────────────────────────────
//  struct Noeud { double x,y; }      Xc[], Yc[] arrays inside FEMState
//  struct Triangle { int i,j,k }     vector<array<int,3>>  FEMState::tri
//  struct ETA  (std::function)       EM_JS callback  js_eval_eta(x,y)
//  struct RHSF (std::function)       EM_JS callback  js_eval_f(x,y)
//  struct VhElement                  inline coordinate lookup per triangle
//
//  Why ETA/RHSF changed:
//    std::function objects cannot cross the C++<->JS boundary. Emscripten's
//    EM_JS macro defines a C function whose body is JavaScript, allowing the
//    WASM module to call user-typed expressions in the browser at runtime.
//
//  Why VhElement was removed:
//    In the WASM version coordinates are looked up on-the-fly from the
//    pre-built FEMState arrays, reducing memory allocations inside each
//    matrix-vector call.
//
// ─────────────────────────────────────────────────────────────────────────────
//  MESH GENERATION
// ─────────────────────────────────────────────────────────────────────────────
//
//  Original                          WASM equivalent
//  ──────────────────────────────    ─────────────────────────────────────────
//  maillageTR(a,b,N,M, noeuds,       FEMState::build(N,M,a,b,lam,dir)
//             triangles)
//
//  Both use the same row-major node numbering and alternating diagonal
//  pattern for triangles (compare maillageTR body with FEMState::build body:
//  the loop structure and index formulas are character-for-character identical).
//
//  In the WASM port the mesh is pre-computed once and stored in the global
//  g_fem struct, then reused for every matrix-vector call inside BiCGSTAB.
//
// ─────────────────────────────────────────────────────────────────────────────
//  ELEMENT ROUTINES
// ─────────────────────────────────────────────────────────────────────────────
//
//  Original                          WASM equivalent
//  ──────────────────────────────    ─────────────────────────────────────────
//  CalcMatBT(el, b00,b01,b10,b11)    calcBT(tx, ty, ...)         same formula
//  integ_eta(el, eta)                integEta(tx, ty)            same midpoint rule
//  DiffTerm(el, eta, K[6])           diffTerm(tx, ty, K[6])      identical math
//  ReacTerm(el, lam, Ml[6])          reacTerm(tx, ty, lam, Ml[6]) identical
//  symPacked(i,j)                    symIdx(i,j)                 same packing
//
//  No mathematical changes at all.  The only difference is that the WASM
//  versions receive raw coordinate arrays (tx[], ty[]) instead of a VhElement.
//
// ─────────────────────────────────────────────────────────────────────────────
//  MATRIX-VECTOR PRODUCT
// ─────────────────────────────────────────────────────────────────────────────
//
//  Original                          WASM equivalent
//  ──────────────────────────────    ─────────────────────────────────────────
//  matvec(elements, eta, lam, ...)   fem_matvec(V) -- reads global g_fem
//  calcAV(elements, V, W, ...)       not ported (only used for normL2)
//
//  matvec() and calcAV() originally took the element list and functors as
//  explicit arguments.  The WASM version uses the global g_fem struct, so
//  the signature simplifies to fem_matvec(V).
//
//  calcAV() (mass-only) was not needed in the WASM port because L2/H1 norms
//  are displayed on the JS side from umin/umax, not computed in C++.
//
// ─────────────────────────────────────────────────────────────────────────────
//  BICGSTAB SOLVER  [*** critical bug fixed in WASM port ***]
// ─────────────────────────────────────────────────────────────────────────────
//
//  Both versions implement the same BiCGSTAB algorithm from Van der Vorst 1992.
//  The WASM v12 had a subtle bug that the original code does NOT have:
//
//  The p-update step reads:
//    p[i] = r[i] + beta * ( p[i] - omega * ??? )
//
//  '???' must be the OLD A*p from the PREVIOUS iteration.
//  In the original bicgstab() 'v' naturally holds old A*p at that point.
//
//  In the WASM v12 rewrite a variable named 'T' (which held old A*s) was
//  accidentally used instead of 'v_vec' (old A*p):
//
//    WRONG  (v12): p[i] = r[i] + beta*(p[i] - omega * T[i])   // T = old A*s
//    CORRECT(v13): p[i] = r[i] + beta*(p[i] - omega * v_vec[i]) // v_vec=old A*p
//
//  Effect: v12 gave noticeably wrong solutions for non-constant eta.
//  Fix in v13: 'v_vec' is declared outside the loop and persists, always
//  holding the most recently computed A*p before being overwritten.
//
// ─────────────────────────────────────────────────────────────────────────────
//  NORMS  (normL2, normL2Grad)
// ─────────────────────────────────────────────────────────────────────────────
//
//  Not ported into the WASM C++ solver.  After solvePDE() returns, the JS
//  side reads umin/umax and displays them in the colorbar.  Full L2 / H1
//  norms could be added with the same formulas used in normL2/normL2Grad.
//
// ─────────────────────────────────────────────────────────────────────────────
//  BOUNDARY CONDITIONS
// ─────────────────────────────────────────────────────────────────────────────
//
//  Original: onBnd[] bool vector passed to matvec / scdmembre as argument.
//  WASM:     FEMState::onBoundary(s) computed inline (works for rectangular
//            domains; avoids allocating a separate boolean array).
//
//  Both support Dirichlet (u=0 on boundary) and natural Neumann (d_n u=0).
//  For Neumann the WASM enforces lambda >= 1e-6 to prevent a singular system.
//
// ─────────────────────────────────────────────────────────────────────────────
//  VISUALIZATION  (WASM only)
// ─────────────────────────────────────────────────────────────────────────────
//
//  buildPDESurface(sol)  – maps solution coefficients X[k] to 3D geometry:
//    vertex position = (x_k, y_k, X[k])    [Z = solution value as height]
//    vertex color    = heatmap(X[k])        [blue -> cyan -> green -> yellow -> red]
//  The mesh triangles from FEMState::tri are reused directly.

//  JS callbacks
// =============================================================================

// =============================================================================
//  FEM State -- pre-computed mesh, reused across matvec calls
// =============================================================================
struct FEMState {
    int N = 0, M = 0;
    double a = 0, b = 0, lam = 0;
    bool dirichlet = true;
    std::vector<double> Xc, Yc;
    std::vector<std::array<int,3>> tri;

    void build(int n, int m, double a_, double b_, double lam_, bool dir) {
        N=n; M=m; a=a_; b=b_; lam=lam_; dirichlet=dir;
        Xc.resize(N+1);
        double hx = 2.0*a/N;
        for (int i=0; i<=N; i++) Xc[i] = i*hx - a;
        Yc.resize(M+1);
        double hy = 2.0*b/M;
        for (int j=0; j<=M; j++) Yc[j] = j*hy - b;
        int K = 2*N*M;
        tri.resize(K);
        int k=0;
        for (int j=0; j<M; j++) for (int i=0; i<N; i++) {
            int A=(N+1)*j+i, B=(N+1)*j+(i+1), C=(N+1)*(j+1)+i, D=(N+1)*(j+1)+(i+1);
            tri[k++] = {{A,B,D}};
            tri[k++] = {{A,D,C}};
        }
    }

    bool onBoundary(int s) const {
        int ii = s%(N+1), jj = s/(N+1);
        return (ii==0 || ii==N || jj==0 || jj==M);
    }
};
// =============================================================================
//  FEM element routines
// =============================================================================

static void calcBT(const double* tx, const double* ty,
                   double& b00, double& b01, double& b10, double& b11) {
    b00=tx[1]-tx[0]; b01=tx[2]-tx[0];
    b10=ty[1]-ty[0]; b11=ty[2]-ty[0];
}

// Approximation of integral_T eta dA  (midpoint rule at 3 edge midpoints)
static double integEta(const double* tx, const double* ty) {
    double b00,b01,b10,b11;
    calcBT(tx,ty,b00,b01,b10,b11);
    double absdet = fabs(b00*b11 - b01*b10);
    double m01x=0.5*(tx[0]+tx[1]), m01y=0.5*(ty[0]+ty[1]);
    double m12x=0.5*(tx[1]+tx[2]), m12y=0.5*(ty[1]+ty[2]);
    double m02x=0.5*(tx[0]+tx[2]), m02y=0.5*(ty[0]+ty[2]);
    double avg = (js_eval_eta(m01x,m01y)+js_eval_eta(m12x,m12y)+js_eval_eta(m02x,m02y))/3.0;
    return avg * absdet / 2.0;
}

// Packed index for 3x3 symmetric matrix (upper triangle)
static int symIdx(int i, int j) {
    if (i>j){int t=i;i=j;j=t;}
    if(i==0&&j==0)return 0; if(i==0&&j==1)return 1; if(i==0&&j==2)return 2;
    if(i==1&&j==1)return 3; if(i==1&&j==2)return 4; return 5;
}

// Diffusion element matrix K[p] = integral_T eta grad(phi_i).grad(phi_j) dA
static void diffTerm(const double* tx, const double* ty, double K[6]) {
    double b00,b01,b10,b11;
    calcBT(tx,ty,b00,b01,b10,b11);
    double det = b00*b11 - b01*b10;
    if (fabs(det) < 1e-30) { for(int i=0;i<6;i++) K[i]=0; return; }
    double val = integEta(tx,ty);
    double inv00= b11/det, inv01=-b10/det;
    double inv10=-b01/det, inv11= b00/det;
    // Reference gradients of P1 hat functions: phi0=(-1,-1), phi1=(1,0), phi2=(0,1)
    static const double gl[3][2] = {{-1,-1},{1,0},{0,1}};
    double gr[3][2];
    for (int i=0; i<3; i++) {
        gr[i][0] = inv00*gl[i][0] + inv01*gl[i][1];
        gr[i][1] = inv10*gl[i][0] + inv11*gl[i][1];
    }
    int p=0;
    for (int i=0; i<3; i++) for (int j=i; j<3; j++)
        K[p++] = (gr[i][0]*gr[j][0] + gr[i][1]*gr[j][1]) * val;
}

// Reaction element matrix M[p] = lambda * integral_T phi_i*phi_j dA
static void reacTerm(const double* tx, const double* ty, double lam, double Ml[6]) {
    double b00,b01,b10,b11;
    calcBT(tx,ty,b00,b01,b10,b11);
    double absdet = fabs(b00*b11 - b01*b10);
    int p=0;
    for (int i=0; i<3; i++) for (int j=i; j<3; j++)
        Ml[p++] = absdet * lam * (i==j ? 1.0/12.0 : 1.0/24.0);
}
// =============================================================================
//  Matrix-vector product W = A * V
//  Uses pre-computed g_fem mesh.
//  Dirichlet BCs: boundary rows become identity (W[k] = V[k]).
// =============================================================================
static std::vector<double> fem_matvec(const std::vector<double>& V) {
    int G = (g_fem.N+1)*(g_fem.M+1);
    std::vector<double> W(G, 0.0);

    for (const auto& t : g_fem.tri) {
        int s[3] = {t[0], t[1], t[2]};
        double tx[3], ty[3];
        for (int k=0; k<3; k++) {
            int ii=s[k]%(g_fem.N+1), jj=s[k]/(g_fem.N+1);
            tx[k]=g_fem.Xc[ii]; ty[k]=g_fem.Yc[jj];
        }
        double KD[6], KR[6];
        diffTerm(tx,ty,KD);
        reacTerm(tx,ty,g_fem.lam,KR);
        for (int i=0; i<3; i++) {
            double res=0.0;
            for (int j=0; j<3; j++)
                res += V[s[j]] * (KD[symIdx(i,j)] + KR[symIdx(i,j)]);
            W[s[i]] += res;
        }
    }

    // Dirichlet BC: replace boundary rows with identity
    if (g_fem.dirichlet) {
        for (int s=0; s<G; s++) {
            if (g_fem.onBoundary(s)) W[s] = V[s];
        }
    }
    return W;
}

// =============================================================================
//  Right-hand side  B[k] = integral_Omega f * phi_k dA
//  Midpoint quadrature at 3 edge midpoints per triangle.
//  Dirichlet: B[k] = 0 on boundary.
// =============================================================================
static std::vector<double> fem_rhs() {
    int G = (g_fem.N+1)*(g_fem.M+1);
    std::vector<double> B(G, 0.0);

    for (const auto& t : g_fem.tri) {
        int s[3] = {t[0], t[1], t[2]};
        double tx[3], ty[3];
        for (int k=0; k<3; k++) {
            int ii=s[k]%(g_fem.N+1), jj=s[k]/(g_fem.N+1);
            tx[k]=g_fem.Xc[ii]; ty[k]=g_fem.Yc[jj];
        }
        double b00,b01,b10,b11;
        calcBT(tx,ty,b00,b01,b10,b11);
        double aire = fabs(b00*b11 - b01*b10) / 2.0;
        double m01x=(tx[0]+tx[1])/2, m01y=(ty[0]+ty[1])/2;
        double m12x=(tx[1]+tx[2])/2, m12y=(ty[1]+ty[2])/2;
        double m02x=(tx[0]+tx[2])/2, m02y=(ty[0]+ty[2])/2;
        double f01=js_eval_f(m01x,m01y), f12=js_eval_f(m12x,m12y), f02=js_eval_f(m02x,m02y);
        // phi_0 nonzero on edges 0-1 (val=1/2) and 0-2 (val=1/2), zero on 1-2
        // phi_1 nonzero on edges 0-1 (val=1/2) and 1-2 (val=1/2), zero on 0-2
        // phi_2 nonzero on edges 1-2 (val=1/2) and 0-2 (val=1/2), zero on 0-1
        B[s[0]] += aire * (f01 + f02) / 6.0;
        B[s[1]] += aire * (f01 + f12) / 6.0;
        B[s[2]] += aire * (f12 + f02) / 6.0;
    }

    if (g_fem.dirichlet) {
        int G2 = (g_fem.N+1)*(g_fem.M+1);
        for (int s=0; s<G2; s++)
            if (g_fem.onBoundary(s)) B[s] = 0.0;
    }
    return B;
}
// =============================================================================
//  BiCGSTAB  --  solves A x = b  using pre-computed g_fem
//
//  KEY FIX vs v12: 'v_vec' persists across iterations, always holding the
//  CURRENT A*p from the previous call to v_vec = fem_matvec(p).
//  The p-update formula needs the OLD A*p, which is exactly what v_vec holds
//  at the start of each iteration (before we overwrite it).
//
//  Old v12 code used T (= old A*s) which was WRONG.
// =============================================================================
static std::vector<double> fem_bicgstab(const std::vector<double>& b_rhs,
                                         double tol, int maxIter)
{
    int n = (int)b_rhs.size();
    std::vector<double> x(n, 0.0);
    std::vector<double> r(b_rhs);       // r = b - A*x0 = b  (x0 = 0)
    std::vector<double> r_hat(b_rhs);   // shadow residual, arbitrary; standard: r_hat = r0

    double normb = 0;
    for (int i=0; i<n; i++) normb += b_rhs[i]*b_rhs[i];
    normb = sqrt(normb);
    if (normb < 1e-30) {
        js_log("BiCGSTAB: RHS norm is zero => trivial solution");
        return x;
    }

    // v_vec persists: always holds the most recently computed A*p
    std::vector<double> p(n, 0.0), v_vec(n, 0.0);
    double rho_old = 1.0, alpha = 1.0, omega = 1.0;

    for (int iter = 0; iter < maxIter; iter++) {

        double rho = 0;
        for (int i=0; i<n; i++) rho += r_hat[i]*r[i];
        if (fabs(rho) < 1e-300) { js_log("BiCGSTAB: rho breakdown"); break; }

        // Update p.
        // iter==0: beta=0 => p = r  (v_vec is zero, doesn't matter)
        // iter >0: v_vec is the OLD A*p from the previous iteration -- CORRECT
        double beta = (iter == 0) ? 0.0 : (rho / rho_old) * (alpha / omega);
        for (int i=0; i<n; i++)
            p[i] = r[i] + beta * (p[i] - omega * v_vec[i]);

        // Compute v_vec = A*p  (overwrites v_vec; will be used as "old A*p" next iteration)
        v_vec = fem_matvec(p);

        double rv = 0;
        for (int i=0; i<n; i++) rv += r_hat[i]*v_vec[i];
        if (fabs(rv) < 1e-300) { js_log("BiCGSTAB: <r_hat,v> breakdown"); break; }
        alpha = rho / rv;

        // s = r - alpha * v_vec
        std::vector<double> s_vec(n);
        for (int i=0; i<n; i++) s_vec[i] = r[i] - alpha * v_vec[i];

        double norms = 0;
        for (int i=0; i<n; i++) norms += s_vec[i]*s_vec[i];
        if (sqrt(norms) < tol * normb) {
            for (int i=0; i<n; i++) x[i] += alpha * p[i];
            char msg[80]; snprintf(msg,80,"BiCGSTAB converged (s-check) iter=%d",iter+1);
            js_log(msg);
            return x;
        }

        // t = A*s
        auto t_vec = fem_matvec(s_vec);

        double ts=0, tt=0;
        for (int i=0; i<n; i++) { ts += t_vec[i]*s_vec[i]; tt += t_vec[i]*t_vec[i]; }
        if (fabs(tt) < 1e-300) { js_log("BiCGSTAB: tt breakdown"); break; }
        omega = ts / tt;

        for (int i=0; i<n; i++) x[i] += alpha*p[i] + omega*s_vec[i];
        for (int i=0; i<n; i++) r[i]  = s_vec[i] - omega*t_vec[i];

        double normr = 0;
        for (int i=0; i<n; i++) normr += r[i]*r[i];
        if (sqrt(normr) < tol * normb) {
            char msg[80]; snprintf(msg,80,"BiCGSTAB converged iter=%d",iter+1);
            js_log(msg);
            return x;
        }

        rho_old = rho;
    }
    js_log("BiCGSTAB: reached max iterations");
    return x;
}
// =============================================================================
//  PDE Surface Geometry -- vertices at (x_k, y_k, X[k])
//  Each node height = FEM coefficient = u_h(node k)
//  Together the mesh represents u_h(x,y) = sum_k X[k] * phi_k(x,y)
// =============================================================================

// Blue -> Cyan -> Green -> Yellow -> Red heatmap
static void fem_colormap(float t, float& r, float& g, float& b) {
    t = std::max(0.f, std::min(1.f, t));
    if      (t < 0.25f) { float s=t/0.25f;          r=0.f; g=s;     b=1.f;   }
    else if (t < 0.50f) { float s=(t-0.25f)/0.25f;  r=0.f; g=1.f;   b=1.f-s; }
    else if (t < 0.75f) { float s=(t-0.50f)/0.25f;  r=s;   g=1.f;   b=0.f;   }
    else                { float s=(t-0.75f)/0.25f;  r=1.f; g=1.f-s; b=0.f;   }
}

static void buildPDESurface(const std::vector<double>& sol) {
    if (g_pdeTriVbo)  { glDeleteBuffers(1,&g_pdeTriVbo);  g_pdeTriVbo =0; g_pdeTriVtxCnt =0; }
    if (g_pdeWireVbo) { glDeleteBuffers(1,&g_pdeWireVbo); g_pdeWireVbo=0; g_pdeWireVtxCnt=0; }

    double umin = *std::min_element(sol.begin(), sol.end());
    double umax = *std::max_element(sol.begin(), sol.end());
    double urange = (umax > umin + 1e-12) ? (umax - umin) : 1.0;

    char dbg[128];
    snprintf(dbg, sizeof(dbg), "buildPDESurface: u=[%.4g, %.4g], %d nodes, %d tris",
             umin, umax, (int)sol.size(), (int)g_fem.tri.size());
    js_log(dbg);

    std::vector<float> tris, lines;
    tris.reserve(g_fem.tri.size() * 3 * 6);
    lines.reserve(g_fem.tri.size() * 6 * 6);

    for (const auto& t : g_fem.tri) {
        int s[3] = {t[0], t[1], t[2]};
        float vx[3], vy[3], vz[3], vr[3], vg[3], vb_[3];
        for (int k=0; k<3; k++) {
            int ii=s[k]%(g_fem.N+1), jj=s[k]/(g_fem.N+1);
            vx[k] = (float)g_fem.Xc[ii];
            vy[k] = (float)g_fem.Yc[jj];
            vz[k] = (float)sol[s[k]];   // height = FEM coefficient X[k]
            fem_colormap((float)((sol[s[k]]-umin)/urange), vr[k], vg[k], vb_[k]);
        }
        // Filled triangle: 3 vertices, each with position + color
        for (int k=0; k<3; k++)
            tris.insert(tris.end(), {vx[k], vy[k], vz[k], vr[k], vg[k], vb_[k]});
        // Wireframe edges (darker)
        const float dk = 0.6f;
        for (int e=0; e<3; e++) {
            int e2=(e+1)%3;
            lines.insert(lines.end(),{
                vx[e],  vy[e],  vz[e],  vr[e]*dk,  vg[e]*dk,  vb_[e]*dk,
                vx[e2], vy[e2], vz[e2], vr[e2]*dk, vg[e2]*dk, vb_[e2]*dk
            });
        }
    }

    g_pdeTriVtxCnt = (int)tris.size()/6;
    glGenBuffers(1,&g_pdeTriVbo);
    glBindBuffer(GL_ARRAY_BUFFER,g_pdeTriVbo);
    glBufferData(GL_ARRAY_BUFFER,(GLsizeiptr)(tris.size()*sizeof(float)),tris.data(),GL_DYNAMIC_DRAW);

    g_pdeWireVtxCnt = (int)lines.size()/6;
    glGenBuffers(1,&g_pdeWireVbo);
    glBindBuffer(GL_ARRAY_BUFFER,g_pdeWireVbo);
    glBufferData(GL_ARRAY_BUFFER,(GLsizeiptr)(lines.size()*sizeof(float)),lines.data(),GL_DYNAMIC_DRAW);
}

//  Exported functions

extern "C" {

    EMSCRIPTEN_KEEPALIVE
    void setSubdivision(float a, float b, int N, int M, float h) {
        buildSubdivision(a, b, N, M, h);
    }

    // solvePDE: called from JS after setting window._fem_f and window._fem_eta
    // dirichlet=1: homogeneous Dirichlet BCs (u=0 on boundary)
    // dirichlet=0: homogeneous Neumann BCs (natural; requires lambda>0)
    EMSCRIPTEN_KEEPALIVE
    void solvePDE(float a, float b, int N, int M, float lambda, int dirichlet) {
        if (g_pdeTriVbo)  { glDeleteBuffers(1,&g_pdeTriVbo);  g_pdeTriVbo =0; g_pdeTriVtxCnt =0; }
        if (g_pdeWireVbo) { glDeleteBuffers(1,&g_pdeWireVbo); g_pdeWireVbo=0; g_pdeWireVtxCnt=0; }

        if (a<=0.f || b<=0.f || N<=0 || M<=0) {
            js_setPDEStatus("Cleared.");
            return;
        }

        double lam = (double)lambda;
        // Neumann requires lambda>0 to avoid singular system
        if (!dirichlet && lam < 1e-6) lam = 1e-6;

        char msg[128];
        snprintf(msg, sizeof(msg),
            "solvePDE: a=%.2f b=%.2f N=%d M=%d lam=%.4g %s",
            (double)a,(double)b,N,M,lam,dirichlet?"Dirichlet":"Neumann");
        js_log(msg);

        js_setPDEStatus("<span style='color:#80d0ff'>Building mesh...</span>");
        // Pre-compute mesh once -- reused by every matvec call in BiCGSTAB
        g_fem.build(N, M, (double)a, (double)b, lam, (bool)dirichlet);

        js_setPDEStatus("<span style='color:#80d0ff'>Assembling RHS...</span>");
        auto B = fem_rhs();

        js_setPDEStatus("<span style='color:#80d0ff'>Running BiCGSTAB...</span>");
        auto sol = fem_bicgstab(B, 1e-8, 2000);

        js_setPDEStatus("<span style='color:#80d0ff'>Building surface...</span>");
        buildPDESurface(sol);

        double umin = *std::min_element(sol.begin(), sol.end());
        double umax = *std::max_element(sol.begin(), sol.end());
        snprintf(msg, sizeof(msg),
            "<b style='color:#80e8a0'>Solved OK</b> &nbsp;%s<br>"
            "%d nodes &nbsp;&middot;&nbsp; %d triangles<br>"
            "u &isin; [%.4g,&thinsp;%.4g]",
            dirichlet ? "Dirichlet" : "Neumann",
            (N+1)*(M+1), 2*N*M, umin, umax);
        js_setPDEStatus(msg);
    }

    EMSCRIPTEN_KEEPALIVE
    void clearPDE() {
        if (g_pdeTriVbo)  { glDeleteBuffers(1,&g_pdeTriVbo);  g_pdeTriVbo =0; g_pdeTriVtxCnt =0; }
        if (g_pdeWireVbo) { glDeleteBuffers(1,&g_pdeWireVbo); g_pdeWireVbo=0; g_pdeWireVtxCnt=0; }
        js_setPDEStatus("Cleared.");
    }
